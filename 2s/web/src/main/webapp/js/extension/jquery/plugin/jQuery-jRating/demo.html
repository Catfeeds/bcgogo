jRating plugin(原版是针对PHP设计的,现在的源码改为部分通用)

参数介绍:

requestPath:(默认:php地址) - 请求地址

showRateInfo:(默认:true) - 是否显示鼠标跟随提示信息

bigStarsPath:(默认:'icons/stars.png') - 大星星图片的地址(stars.png).

smallStarsPath:(默认:'icons/small.png') - 小星星图片的地址(small.png).

type:(默认:'big') - 评价样式的类型.. 可以设置为小星星'small'或者大星星'big'.

step:(默认:false) - 是否一整个一整个星星选择.

isDisabled:(默认:false) - 是否禁用评价功能

length:(默认:5) - 显示的星星的数量.

decimalLength:(默认:0) - 评价分数的小数点精度

rateMaxr:(默认:20) - 最大的评价分数(即满分)

rateInfosX:(默认:45) - 提示信息的绝对定位X坐标.

rateInfosY:(默认:5) - 提示信息的绝对定位Y坐标.
Exemple 1 (基本应用(带请求地址)) : 
  $(document).ready(function(){
    $(".bacic").jRating({
    	requestPath:'test.do?method=submitRating'
	});
  });
Exemple 2 (type : small - average 10 - id 2 - 40 stars) : 
  $(document).ready(function(){
    $(".exemple2").jRating({
	  type:'small', // 评价样式的类型.. 可以设置为小星星'small'或者大星星'big'
	  length : 40, // 总共的星星数量
	  decimalLength : 1, // 小数点的精度
      requestPath:'test.do?method=submitRating'
    });
  });
Exemple 3 (step : true - average 18 - id 3 - 15 stars) : 
  $(document).ready(function(){
    $(".exemple3").jRating({
	  step:true, // no half-star when mousemove
	  length : 20, // nb of stars
	  decimalLength:0, // number of decimal in the rate
      requestPath:'test.do?method=submitRating'
    });
  });
Exemple 4 (禁用评价) : 
  $(document).ready(function(){
    $(".exemple4").jRating({
	  isDisabled : true
	});
  });
Exemple 5 (当请求成功的onSuccess()方法和当请求发生错误的onError()方法) : 
  $(document).ready(function(){
    $(".exemple5").jRating({
      requestPath:'test.do?method=submitRating',
	  length:10,
	  decimalLength:1,
	  onSuccess : function(){
	    alert('Success : your rate has been saved :)');
	  },
	  onError : function(){
	    alert('Error : please retry');
	  }
	});
  });
Exemple 6 (不显示提示信息) : 
  $(document).ready(function(){
    $(".exemple6").jRating({
	  length:10,
	  decimalLength:1,
	  showRateInfo:false,
      requestPath:'test.do?method=submitRating'
	});
  });
Exemple 7 (设置初始化星星的评分) : 只要在rateInit上写明就可以了,rateInit="10_1",
其中_前面的表示你要显示星星时默认的分值average,这里的average是10,即选中的分值10分的星星,这里默认满分是20分,一颗大星就是2分(也可以是小数,如:rateInit="0.5_5").
例如: 这里有10颗星星,写上20就可以全选中.
_后面的参数只是一个数字的id,方便动态传值和绑定值 
	rateInit="10_5"
